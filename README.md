# Landing — Simple HTML & CSS Project

A clean, responsive landing page built with plain HTML and CSS. This repository is a small frontend project for practicing layout, responsive design, and asset organization.

**Preview**

- Open `index.html` in your browser to view the page.

**Highlights**

- Single-page layout with header, hero, feature cards, quote, call-to-action, and footer.
- Responsive CSS in `styles/styles.css`.
- Assets organized under `assets/images`.

**Project Structure**

- `index.html` — Main HTML file.
- `styles/styles.css` — Main stylesheet.
- `assets/images/` — Image assets used on the page.
- `LICENSE` — Project license.

**How to view locally**

1. Open the `index.html` file directly in your browser (double-click or choose "Open File").

2. VS Code: Install the "Live Server" extension and click "Go Live".
